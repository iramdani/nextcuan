Pengaturan Moota sekarang disederhanakan menjadi 2 field wajib: Webhook URL HTTPS dan Secret Token untuk verifikasi signature HMAC.

Gunakan URL webhook publik dari Cloudflare Worker atau proxy Anda.

Catatan penggunaan

1. Moota mengirim payload JSON array via `POST` dan signature di header `Signature`.

2. Arahkan webhook ke endpoint Cloudflare Worker.

3. Jalankan `Test Koneksi Moota` sebelum menyimpan.

Isi public key, URL endpoint, dan private key langsung dari aplikasi.
